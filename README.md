## Project Overview
This is a beginner-friendly Python mini project. It helps the user add tasks, view tasks, delete tasks, and clear all tasks.

## Concepts Used
- Variables
- Lists
- Functions
- Loops
- If-else statements
- File handling
- JSON

## Features
1. Add task
2. View tasks
3. Delete task
4. Clear all tasks
5. Save tasks in file

## Files Included
- `todo_list.py`
- `tasks.json`
- `documentation.txt`
- `README.md`

## How to Run
```bash
python todo_list.py
```
